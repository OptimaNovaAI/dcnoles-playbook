# DC Noles President's Transition Playbook

The 2026-27 board transition playbook for the Seminole Club of Greater Washington, DC. A single self-contained page covering the state of the handoff, a 10-day catch-up sprint, the account & access checklist, the new president's first 30 days, the bylaws governance calendar, and inherited assets and relationships.

**Live site:** https://dashboards.optimanova.ai/dcnoles-playbook/

This is a static, single-file site — everything (markup, styles, and a small vanilla-JS handoff checklist that persists to `localStorage`) lives in `index.html`. There is no build step and no server-side code.
